# TweetApp

## サイト概要
Tweet投稿サイト

### テーマを選んだ理由
Ruby on Railsの基礎学習としてCRUDアプリケーションの制作を行う。

### 機能
- 投稿一覧
- 投稿詳細
- 新規投稿
- 投稿編集
- 投稿削除
